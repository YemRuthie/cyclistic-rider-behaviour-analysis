# 🚴 Cyclistic Rider Behaviour Analysis

## Overview

This project was completed as part of the Google Data Analytics Professional Certificate.

The objective was to understand how annual members and casual riders use Cyclistic bikes differently and provide data-driven recommendations to increase membership conversions.


## Business Task

Analyse behavioural differences between casual riders and annual members and identify opportunities to convert casual riders into long-term members.


## Tools Used

- Python
- Pandas
- Tableau Public
- GitHub


## Project Workflow

1. Data Cleaning
2. Feature Engineering
3. Exploratory Data Analysis
4. Data Visualisation
5. Business Recommendations


## Key Findings

- Casual riders take trips **58% longer** than members.
- Casual riders are most active on weekends.
- Members exhibit commuter behaviour, with ride peaks around **8 AM** and **5 PM**.
- Both rider groups strongly prefer electric bikes.
- Casual riders primarily use Cyclistic for leisure, while members use it for routine transportation.


## Business Recommendations

- Target frequent weekend riders with membership promotions.
- Introduce commuter challenges to encourage weekday riding.
- Position membership as a transportation solution rather than a leisure product.
- Promote member-exclusive benefits for electric bikes.
